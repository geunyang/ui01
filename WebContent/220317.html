html: Hyper Markup Language
웹브라우저로 표현하고 실행하기 위한 웹문서를 만드는 언어
<>:부등호기호로 둘러싸인 요소를 태그라 한다

elements(=요소):html 문서를 구성하는 약속된 단어

style:웹문서를 보기좋게 꾸미는데 사용
css:cascading style sheet 웹문서를 보기좋게 꾸미는 디자인표현(형용사/부사)
javascript/jQuery 프로그래밍적 요소로 사용자의 행위에 의한 동작(동사)

1.블록레벨텍스트태그

혼자서 한줄을차지하여 너비가 100% 꽉차게 표현되는 태그

h태그 : h1>h2>h3>h4>h5>h6 - heading

p태그 : 단락을 표현하는 태그

hr태그 : 주제가 바뀌거나 분위기 전환을 할때 사용하는 가로선



주석 : 화면에 보이지는 않게 내부적으로 설명을 넣고자 할때 사용하는 것
div 영역을 나누는 블록태그

2.인라인 레벨 텍스트 태그

컨텐츠만큼의 너비를 차지하는 태그

strong태그 굵게 표시하는 태그
span태그:텍스트의 일부만 특정스타일처리 할때 사용

목록

ul(unodered list) : 순서 없는 목록(사용빈도높음)

ol(odered list) : 순서 있는 목록

  • 디스크1
  • 디스크2
  • 디스크3

ul의 list-style-type이 기본 disk이다

  1. 메뉴
  2. 메뉴
  3. 메뉴

ol의 list-style-type이 기본 decimal(정수)이다.
숫자, 알파벳, 로마숫자로 순서를 표시한다. 설정변경 : type
시작값을 지정 가능 : start

4. 표를 만드는 태그

table 태그, tr(table row) 태그, td(table data) 태그
행열로 이루어진 표의 형태를 만들때 사용하는 태그
tr : 행에 해당하는 태그
td : 행 내부의 열에 해당하는 셀 태그
th(table head) : 행 내부의 제목에 해당하는 셀 태그 rowspan='number' : 열병합 colspan='number' : 행병합
